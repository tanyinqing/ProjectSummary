# 1. 软件常用的快捷键  

> ## <h1 id="0101">1. studio常用的快捷键</h1>

> ## [返回目录](#01) 

>> ## 1       常用的快捷键
>> 
>> ## Ctrl+Shift+Up/Down 代码向上/下移动。
>> ## Ctrl+X 删除行
>> ## Ctrl＋F12，可以显示当前文件的结构
>> ## Ctrl+D 复制行
>> ## Alt+ Up/Down 在方法间快速移动定位
>> ## Ctrl+H 显示类结构图
>> ## Shift + Enter  光标在一行代码的任何位置，使用这个方法就会自动跳转下一行，不用将光标移到行尾；
>>  
>> ## 2       应该熟练的快捷键
>>  
>> ## Ctrl+J  模板代码提示    
>> ## Ctrl+Alt+T：选中一块代码，按此组合键，可快速添加if 、for、try/catch等语句。
>> ##  Ctrl＋Shift＋Backspace可以跳转到上次编辑的地方
>> ##  Ctrl+N   查找类
>> ##  Ctrl+Shift+N 查找文件
>> ##  Ctrl+Alt+L  格式化代码
>> ## Ctrl+Alt+F7:相当于在工程中查找该关键词
>> ## Ctrl+Tab：打开界面切换窗口，保持按住Ctrl键，选中相应的要打开的窗口。
>> ## Ctrl+W：选中光标所在的所在的单词（一个成员变量或者是一个方法名），多按一次会选中所在的语句，
>> ##   再多按一次会选中所在的代码块。。。依次类推，每增加一次会扩大一级选中的范围。
>> ##   Ctrl＋E，可以显示最近编辑的文件列表
>> ##  Shift＋Click可以关闭文件
>> ##  Ctrl＋[或]可以跳到大括号的开头结尾
>> ##  Ctrl＋Shift＋Backspace可以跳转到上次编辑的地方
>> ##  Ctrl+Alt+ left/right 返回至上次浏览的位置
